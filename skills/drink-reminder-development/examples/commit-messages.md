# Commit Message Examples

## Feature Commits
```
feat(controllers): Add new reminder controller
```
```
feat(views): Implement custom reminder sound support
```
```
feat(models): Add weekly statistics tracking
```

## Bug Fix Commits
```
fix(models): Fix date comparison in data loading
```
```
fix(controllers): Resolve infinite loop in reminder check
```
```
fix(views): Fix window positioning on multi-monitor setups
```

## Documentation Commits
```
docs(readme): Update installation instructions for Windows
```
```
docs(architecture): Add MVC design pattern explanation
```
```
docs(api): Add type hints documentation
```

## Refactoring Commits
```
refactor: Extract time parsing to utility function
```
```
refactor(controllers): Simplify reminder scheduling logic
```
```
refactor(models): Improve data validation
```

## Test Commits
```
test(controllers): Add integration tests for reminder flow
```
```
test(models): Add unit tests for data persistence
```
```
test: Fix flaky time-related tests
```

## Chore Commits
```
chore(deps): Update tkinter to latest version
```
```
chore(config): Add new logging configuration options
```
```
chore: Update .gitignore for new data files
```

## Performance Commits
```
perf(models): Optimize JSON loading with caching
```
```
perf(controllers): Reduce CPU usage in main loop
```

## CI/CD Commits
```
ci: Add automated testing workflow
```
```
ci: Configure GitHub Actions for linting
```

## Build Commits
```
build: Update pyproject.toml dependencies
```
```
build: Add Windows installer generation
```

## Revert Commits
```
revert: "feat(views): Add dark mode support"
```
```
revert: "refactor: Simplify data model"
```
